구현하지 못한 내용 - 1인칭 시점, 충돌처리

구현한 내용 - 나머지

기타 실행시 알아야할 요소들 
- depthtest를 하면 객체가 짤리는 현상 발생
- 미로는 고정되어 있으며 충돌처리가 없음

키보드 명령어
========================================================================
o/p: 투영을 선택한다. (o:직각투영/ p:원근투영)(기본은 원근투영)
z/Z: 원근 투영 시 z축으로 이동할 수 있게 한다.
m/M: 육면체들이 위아래로 움직인다./ 멈춘다.
y/Y: 카메라가 바닥의 y축을 기준으로 양/음 방향으로 회전한다. 한번더 누르면 멈춘다.
r: 미로를 제작한다.
v: 육면체들 움직임이 멈추고 낮은 높이로 변한다 다시 누르면 움직임이 다시 시작된다
s: 미로에서 객체가 나타난다. -> 객체를 소환하면 h(depthtest)를 눌러주세요
4/6/8/5 좌/우/상/하로 객체가 이동한다.
+/-: 육면체 이동하는 속도 증가/감소;
1/3: 카메라 시점 1인칭(*구현 실패*)/3인칭 변환
c: 모든 값 초기화
q: 프로그램 종료
 =================================기타====================================
h: depth테스트 끄기/켜기(객체가 많이 깨지니 미로할때만 사용
j/l/i/k: 좌/우/상/하로 1인칭 시점 (실패한) 화면 이동 1번을 눌러야 사용가능
